# /research "sujet de recherche"

Recherche des meilleures solutions algorithmiques actuelles.

## Étape 1: Définition

Clarifier le sujet de recherche avec l'utilisateur:
- Quel problème résoudre?
- Quelles contraintes (mémoire, CPU, latence)?
- Quels critères de succès?

## Étape 2: Recherche Internet

Utiliser MCP Brave pour rechercher:
- "[sujet] algorithm 2024 2025"
- "[sujet] rust implementation"
- "[sujet] benchmark comparison"

## Étape 3: Recherche Académique

Rechercher sur arXiv:
- Papers récents (< 2 ans)
- Avec implémentations
- Avec benchmarks

## Étape 4: Synthèse

Invoquer @research-algo pour:
1. Analyser les résultats
2. Comparer les options
3. Créer le document .research/YYYY-MM-DD-sujet.md

## Étape 5: Recommandation

Présenter:
- Top 3 solutions identifiées
- Avantages/inconvénients
- Recommandation finale
- Plan d'implémentation suggéré
